4) creating a repository in GitHub using SSH

copy the path it is in above like c://

copy that public key link
login into GitHub-->go to setting-->select SSH and GPG keys
click new SSh key --->give title then paste copied public key
click new repository -->give name
click on SSh protocal
copy the link looks like git remote add origin
execute that
